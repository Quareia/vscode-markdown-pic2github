1个vscode插件，提供markdown的github图片外链。编写markdown文件时可以上传本地图片并拷贝到本地git仓库，最后自动提交到github

### Usage
- settings.json中修改对应的属性
    1. `local_git_folder`：本地存储图片的git仓库文件夹名，为绝对路径
    2. `remote_github_repo`：github对应的图片仓库
- `ctrl+alt+g`：选择上传图片
